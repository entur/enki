# Cookie Information based cookies consent management

## Enabling the feature

Once enabled in bootstrap.json

    "sandboxFeatures": {
        "CookieInformation": true
    }

CookieInformation feature can be wired in as:

    <ComponentToggle
        feature="CookieInformation"
    />
